#  StockTrack App

#### Libraries and SDK's

1. **UIKit** - User Interface
2. **CoreData** - Persistence
3. **Alamofire & RxAlamofire** - Networking
4. **RxSwift** - Dependency on RxAlamofire
5. **Charts** - Line Chart
6. **Cocoa Pods** - Dependecy Management
